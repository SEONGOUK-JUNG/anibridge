3🆘 응급 카드 — 이 앱의 핵심

한밤중에 아이가 아파서 처음 가는 병원에 갔을 때, 수의사 선생님은 이런 걸 묻습니다: "몇 살이에요? 접종은요? 먹는 약 있어요? 알레르기는요? 최근에 어땠어요?" — 당황한 상태에선 기억이 잘 안 나죠.

응급 카드는 병원 문진표와 같은 순서로 정리되어 있습니다:
품종·나이·체중·혈액형 → 주거 형태 → 동거견/묘 → 식이 → 음수량 → 배뇨·배변 횟수 → 현재 컨디션 → 병력·수술 → 복용 약물·보조제 → 약·백신 과민 반응 → 기초·추가 접종 → 심장사상충·외부기생충 예방 → 검사자료 여부 → 접종 기록 → 최근 진료 → 최근 30일 증상

사용법

  1. 홈 → 🆘 응급 카드 (또는 케어수첩 맨 위, MY 메뉴)
  2. 카드 맨 위의 ✏️ 문진 정보 작성 · 수정하기 버튼으로 문진 항목을 채워두세요 — 한 번만 작성하면 계속 유지됩니다
  3. 병원에서 화면을 그대로 보여주거나, 카카오톡 공유 / 텍스트 복사로 전달
  4. 급할 땐 카드 안의 🌙 지금 문 연 24시 병원 찾기 버튼으로 바로 병원 검색
아니의 꿀팁접종 기록·최근 진료·최근 증상은 케어수첩 기록으로 자동 작성돼요. 평소에 기록만 하면 준비 끝!

4🩺 건강 지키기 도구들

🚨 건강 시그널

증상이 얼마나 급한지 3단계로 알려줍니다: 🔴 지금 즉시 병원(호흡곤란·경련·소변 못 봄 등) / 🟡 24시간 내 진료 / 🟢 기록하며 관찰. 🔴 항목에 해당되면 버튼 한 번으로 24시 병원 검색으로 넘어갑니다.

🩺 검진 체크리스트

우리 아이의 종(강아지/고양이)과 나이(어린이/성체/노령)에 맞는 건강검진 항목을 자동으로 보여줍니다. 항목을 탭해서 완료 체크하면 저장돼요.

🏥 내 주변 병원

펫플레이스 탭에서 📍 현재 위치로 찾기를 누르면 가까운 동물병원이 거리순으로 나옵니다. 🌙 24시 필터로 심야 병원만 골라 볼 수 있고, 📞 버튼으로 바로 전화, Ⓝ 버튼으로 네이버 리뷰 확인이 가능합니다. 동반 카페·식당·미용실 검색도 같은 방식이에요.

🍇 먹어도 되나요? (위험 음식 사전)

음식 이름을 검색하면 강아지/고양이별 위험도가 나옵니다: 🔴 위험 / 🟡 주의 / 🟢 괜찮음 + 대처법. "건포도", "초코파이"처럼 별칭으로 검색해도 찾아줍니다.

브리의 경고⚠️ 포도·초콜릿·양파·자일리톨(무설탕 껌)은 소량도 위험! 먹었다면 검색 결과의 병원 찾기 버튼으로 즉시 병원에 연락하세요.

탭기록 내용

🏥 병원기록병원명·진단·비용

📝 메모자유 기록
아니의 꿀팁📋 사전 문진서: 병원 예약 후 이상증상 탭의 "사전 문진서로 내보내기"를 누르면 진료에 필요한 정보가 정리돼요. 진료 시간이 짧아져요!

6🍚 식단·급여량 관리

  1. 케어수첩 → 식단 탭에서 사료를 등록합니다 — 📷 사료 봉투를 찍으면 AI가 제품명과 칼로리를 자동으로 읽어줘요 (또는 봉투 뒷면의 kcal/kg 직접 입력)
  2. 체중·나이·중성화 여부로 하루 권장량이 자동 계산됩니다 (수의 영양학 표준 공식) — 예: "하루 71g, 2회 × 36g씩, 간식은 27kcal 이내"
  3. 먹일 때마다 g 입력 후 🍚 식사 또는 🦴 간식 버튼
  4. 게이지로 오늘 상태 확인: 적정 ✅ / 부족 / 과식 ⚠️ — 간식이 하루 칼로리의 10%를 넘으면 경고

식사를 기록하면 홈의 "오늘 챙기기" 체크리스트에 자동으로 ✅ 표시됩니다. 복용약이 프로필에 있으면 💊 약 챙기기 항목도 자동 생성 — 탭해서 체크하세요.

7🐾 산책(강아지) · 🧶 놀이(고양이)

수의학적으로 강아지에게는 산책이, 고양이에게는 실내 놀이가 필요합니다. AniBridge는 등록된 아이의 종에 따라 화면이 자동으로 바뀝니다.

🐶 강아지로 등록했다면 — 산책 모드

  1. 홈 또는 산책 탭에서 🐾 산책 시작 — 위치 권한을 허용하면 GPS로 실제 거리를 측정합니다
  2. 걷는 동안 "경로 N개 기록 중"이 표시됩니다 (화면을 켠 채로 걸어주세요)
  3. ⏹️ 산책 종료를 누르면 저장 완료 — 하루 목표는 30분
🗺️ 발자국 지도 (강아지 전용): 산책 탭에 최근 경로 지도가 바로 보입니다. 지도를 탭하면 크게 열리면서 — 로고의 강아지와 고양이가 실제 걸은 길을 따라 걸어가며 빨간 발자국을 남기는 애니메이션이 재생됩니다. 경로는 실제 도로에 맞춰 자동 보정돼요. ▶ 산책 다시 보기로 반복 재생!
아니의 안내📍 경로가 저장되려면 실외에서 50m 이상·1분 이상 걸어야 해요. 실내나 GPS 거부 시엔 시간만 기록됩니다.

🐱 고양이로 등록했다면 — 놀이 모드

  1. 같은 자리의 버튼이 🧶 놀이 시작으로 바뀝니다 — 실내 놀이라서 GPS 없이 바로 타이머가 시작돼요
  2. 낚싯대·레이저·사냥놀이로 신나게 놀아주세요 — 하루 목표는 20분 (10~15분씩 2~3회가 좋아요)
  3. ⏹️ 놀이 종료를 누르면 저장 완료
🧶 놀이 가이드: 이동 거리·경로 지도 대신 놀이에 맞는 화면이 나옵니다 — 권장 놀이 시간, 추천 장난감, 그리고 "놀이 후엔 간식으로 사냥을 마무리해주세요" 같은 고양이 맞춤 팁까지. 주간 기록·리포트·오늘의 루틴도 모두 놀이 기준으로 계산됩니다.
브리의 한마디여러 마리를 키운다면 아이를 전환할 때마다 산책 ↔ 놀이 화면이 자동으로 바뀌어요.

11❓ 자주 묻는 질문

Q정말 다 무료인가요?
네! 광고도 없고 결제도 없습니다. AI 기능만 안정적인 운영을 위해 하루 횟수(각 10회)가 있고, 매일 자정 충전됩니다.
Q내 기록은 어디에 저장되나요?
모두 휴대폰 안에만 저장되며 서버로 전송되지 않습니다. 그래서 백업이 중요해요! (AI 분석 시 보낸 사진·질문만 AI 처리를 위해 일시적으로 전송됩니다)
QAI 분석이 "괜찮다"고 했는데 병원에 안 가도 되나요?
아니요. AI는 사진만으로 판단하는 참고 도구라 놓치는 것이 있을 수 있습니다. 아이가 평소와 다르다면 AI 결과와 관계없이 병원에서 확인하세요.
Q위치 권한은 왜 필요한가요?
산책 거리·경로 측정과 내 주변 병원 검색에만 사용됩니다. 위치 정보는 폰에만 저장돼요.
Q여러 마리를 키워요.
MY에서 최대 5마리까지 등록할 수 있고, 탭으로 전환하면 기록·식단·응급 카드가 각각 관리됩니다.
Q문의는 어디로 하나요?
MY → 고객센터에서 문의를 보내거나, anibridge@naver.com 으로 연락주세요.
