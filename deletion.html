계정 삭제 요청 방법

AniBridge 계정 및 관련 데이터를 삭제하려면 아래 방법 중 하나를 선택하세요.

방법 1: 앱 내에서 직접 삭제

  1. AniBridge 앱을 실행합니다
  2. 하단 메뉴에서 MY 탭을 선택합니다
  3. 카카오 로그인 섹션에서 "로그아웃" 버튼을 누릅니다
  4. 반려동물 카드에서 🗑️ 삭제 버튼을 눌러 등록된 반려동물 정보를 삭제합니다
  5. 앱을 삭제하면 기기에 저장된 모든 데이터가 제거됩니다

방법 2: 이메일로 삭제 요청

서버에 저장된 데이터(카카오 로그인 정보 등)를 완전히 삭제하려면 아래 이메일로 삭제를 요청해 주세요.

📧 koreagwangju@gmail.com

요청 시 포함할 정보:

  • 카카오 닉네임 또는 카카오 계정 이메일
  • 삭제 요청 사유 (선택)

요청 접수 후 영업일 기준 7일 이내에 처리됩니다.
